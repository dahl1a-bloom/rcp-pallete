# 🎨 rcp-palette (СSS Color Parser)

Простий і надійний парсер для перетворення рядкових представлень CSS-кольорів (Hex та RGB) у внутрішню структуру даних Rust.

## Технічний опис процесу парсингу

Проєкт **`rcp-palette`** парсить рядки, які відповідають стандартам CSS Color Module Level 3/4. На поточному етапі підтримуються:

1.  **Повна Hex-форма:** `#RRGGBB` (наприклад, `#1A2B3C`).
2.  **Скорочена Hex-форма:** `#RGB` (наприклад, `#FA0`).
3.  *(Майбутнє)* Функціональна RGB-форма: `rgb(R, G, B)`.

### Процес парсингу

1.  **Диспетчеризація:** Вхідний рядок перевіряється на наявність префіксу (`#`).
2.  **Hex-аналіз:** Якщо знайдено `#`, рядок передається функції `parse_hex`, яка перевіряє довжину (3 або 6 символів).
3.  **Декомпресія (для #RGB):** Трисимвольний код дублюється (наприклад, `#FA0` перетворюється на `#FFAA00`).
4.  **Перетворення:** Кожен двосимвольний компонент (R, G, B) парситься з шістнадцяткової системи числення на десяткове число `u8` (0-255) за допомогою `u8::from_str_radix(..., 16)`.
5.  **Обробка Помилок:** Використовується крейт `thiserror` для точного відображення помилок, таких як недійсна довжина (`InvalidLength`) або не-Hex символи (`ComponentParseError`).

### Використання результатів парсингу

Успішний парсинг повертає структуру `Color { r: u8, g: u8, b: u8 }`. Ця структура є універсальним поданням кольору і може бути використана для:

* **Графічного відображення** (наприклад, у візуалізаторах чи графічних інтерфейсах).
* **Колірних обчислень** (змішування, перетворення в інші моделі).
